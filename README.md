## 🌾 Smart Crop Advisory System

**🏆 Achievement:** 3rd Place – DesignXpo 1.0

### 📌 Problem Statement

Build a smart system to help farmers make better crop decisions using available data and simple AI assistance.

### ⚙️ Solution

A web-based Smart Crop Advisory System that provides:

* Crop recommendations
* Weather-based suggestions
* Basic farming insights for better yield

### 🚫 Constraints

* No agentic AI used
* Only used:

  * ChatGPT
  * Gemini
  * DeepSeek

### 💡 Features

* Clean and simple UI
* Farmer-friendly design
* Advisory based on inputs
* Lightweight and fast

### 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

### 🎯 Outcome

Developed a functional prototype under constraints and secured **3rd place** in DesignXpo 1.0.

---

> Built with limited AI tools, focusing on practical usability and clean design.
